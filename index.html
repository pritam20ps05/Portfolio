Techstack

Bootstrap

Bootstrap is a popular open-source front-end framework that simplifies web development by providing pre-designed components and responsive layouts, helping developers create visually appealing and mobile-friendly websites with ease.

Jquery

jQuery is a fast, lightweight JavaScript library that simplifies web development by providing a wide range of functions and utilities for DOM manipulation, event handling, and AJAX interactions, making it easier for developers to enhance and streamline web applications.

GSAP

GSAP, or the GreenSock Animation Platform, is a robust JavaScript library for creating high-performance, smooth animations and interactive web experiences, offering a powerful set of tools and features for web developers to bring dynamic and engaging content to their websites with ease.

Flask

Flask is a lightweight and flexible Python web framework used for building web applications. It is known for its simplicity and minimalism, providing essential tools and libraries to create web-based projects quickly and efficiently, making it a popular choice among developers for smaller-scale applications and prototypes.

MongoDB

MongoDB is a NoSQL, document-oriented database system, designed to efficiently store and manage data in a flexible, schema-less format. It offers high scalability and performance, making it a popular choice for applications that require quick and dynamic data handling, while also supporting various querying and data modeling capabilities.

Vercel

Vercel is a cloud platform that specializes in deploying and hosting web applications, providing an easy and scalable solution for developers to launch their projects quickly. It offers features like serverless functions, continuous deployment, and global CDN, making it a popular choice for hosting websites and web applications with minimal configuration and maximum performance.
